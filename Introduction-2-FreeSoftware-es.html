Software Libre

Durante todo de este curso, vamos a tratar de utilizar, cuando sea posible, software libre y código abierto.  Software Libre o Código Abierto (los términos se usan indistintamente) es un software que se distribuye bajo una licencia especifica que permite a otros a examinar, modificar, y distribuir sin tener que pedir permiso adicional del autor original.  Esto previene cualquier empresa de controlar el software, y permite a las comunidades grandes de los participantes a trabajar juntos, añadiendo características importantes que hacen que el software útil para ellos mismos y otros.  Debido a que este software es abierto a la comunidad, es también casi siepre se distribuye gratis.  (En Inglés hay uno palabra por gratis y libre es ¨Free¨.)  Todo el software libre utilizado en este curso puede ser libremente descargado desde varios sitios web.

Dicho esto, este curso es principalmente acerca de los aspectos multimedia, no los aspectos del software.  Como tal, también se refiere al uso de no-libre (propietario) herramientas. El método general utilizado en este curso es aprender los conceptos básicos de cada tema utilizando las herramientas de código abierto, entonces una vez que un conocimiento básico del tema se tiene, el curso de u breve tiempo para aprender cómo aplicar esos conceptos en las herramientas para propietario tema.

Herramientas de Código Abierto qeu va a utilizar.  Pase el ratón sobre el icono para leer una breve descripción, haga clic en el icono para ir a la página web de la herramienta.

Además, ara algunos de esta clase, que será el funcionamiento de ordenadores que utilizan el sistema operativo Ubuntu, en lugar de Microsoft Windows.  Ubuntu es un sustituto de código abierto para Microsoft Windows.  Sin embargo, también hace mucho más que Windows, ya que viene por defecto con uch más software, como muchas de las herramientas anteriores, ye casi todos los Software Libre / Código Abierto puede ser instalado en él, simplemente buscando en el ¨Centro de Software¨ y haciendo clic en instalar.

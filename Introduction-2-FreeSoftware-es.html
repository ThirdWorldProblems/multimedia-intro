Software Libre

Durante todo de este curso, vamos a tratar de utilizar, cuando sea posible, software libre y de código abierto.  Software Libre o de Código Abierto (los términos se usan de forma intercambiable) es un software que se distribuye bajo una licencia especifica que permite a otros examinar, modificar, y distribuir el software sin necesitar permiso adicional del autor original.  Esto evita que cualquier empresa privada controle el software, y permite que grandes comunidades de participantes trabajen junto, añadiendo características importantes que hacen que el software sea más útil para ellos mismos y otros.  Debido a que este software está abierto a la comunidad, es también casi siempre se distribuido libre de costo.  (Por eso el nombre de "free software": En inglés la palabra "free" significa tanto "libre" como "gratis".)  Todo el software libre utilizado en este curso puede ser descargado gratuitamente desde varios sitios web.

Dicho esto, este curso es principalmente acerca de aspectos multimedia, no de aspectos de software.  Como tal, también estudiaremos el uso de herramientas no-libres (con derechos propietarios) herramientas. El método general utilizado en este curso es aprender los conceptos básicos de cada tema utilizando las herramientas de código abierto.  Una vez tengamos un conocimiento básico del tema dedicaremos un breve tiempo a aprender cómo aplicar esos conceptos decada tema usando herramientas propietarias.
